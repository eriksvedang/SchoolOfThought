Tempest (a temporary title of this project)

Hello!

This website is an ongoing project for providing a series of exercises for people interested in functional programming. In particular it's designed for those who want to understand how to apply deep and interesting concepts from mathematics when solving practical problems.

Advanced articles about functional programming often use a lot of peculiar words – Profunctor, Monoid and Arrow, just to name a few. Learning about their mathematical properties can be a lot of fun, but how should they be used in actual code? Without a practical understanding it's hard to recognize the patterns and situations where they are helpful.

Hopefully the exercises here can help in that regard! Or if you think they're lacking, please help us out by making a pull request. The code for this website can be found here.

Prerequisites

You need to know a bit of Haskell to really make use of this site. If you'll try to solve the exercises with some other language like Ocaml, F#, Lisp, etc, you will hopefully get something good out of it but things are going to diverge at one point or another.

Also make sure you have a good way of setting up small projects (one interactive file for each exercise should be fine).
